Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The meta element contains metadata for the webpage.
  • The body element represents the visible content shown to the user.
  • The h1-6 elements are used as headers, only use h1 once.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class selectors will assign CSS elements to any HTML element assigned to the class.
  • Similarly to class selectors, Element names can be used to assign CSS properties.

Git

  • Git is a funny version control system that is utilized in tandom with github.
  • git status: a command that lists the current working branch.
  • Branches are used for teams to develop simultaneously. Use git checkout -b new-branch-name to make a new branch and switch to it.
